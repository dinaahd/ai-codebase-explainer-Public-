💻 AI Codebase Explainer

An AI-powered tool that helps developers understand codebases instantly using Large Language Models (LLMs).

🚀 Features
📂 Upload or paste code
📁 Multi-file support (simulate repository understanding)
📘 AI-generated structured explanations
💬 Ask questions about the code (interactive Q&A)
⚡ Fast and intuitive UI built with Streamlit
🧠 What It Does

This tool analyzes code and provides:

Overall purpose of the code
Step-by-step execution flow
Key functions and logic
Suggestions for improvement
Time and space complexity (when applicable)

It also allows users to ask contextual questions about the code, acting like a developer copilot.

🛠️ Tech Stack
Frontend/UI: Streamlit
Backend: Python
AI Model: LLM via OpenRouter API (LLaMA 3.1)
Other Tools: Git, GitHub
